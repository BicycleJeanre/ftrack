# Budget Workflow

## 1.0 Purpose

1.1 Explain how to use the Budget workflow to plan spending, track actuals, and iterate using budgets and projections.

## 2.0 When To Use Budget

2.1 Use Budget when you want.

2.1.1 A budget grid.

2.1.2 Actual Transactions.

2.1.3 Plan vs actual comparisons and variance.

2.2 If you do not need budgets and actuals, consider the General workflow.

## 3.0 Use the Budget Workflow

3.1 Go to Forecast.

3.2 Select the Budget workflow in the left nav.

3.3 Create a scenario.

3.4 Set Period Type to Month in most cases.

3.5 Set Start and End dates for the projection window you want to track.

## 4.0 Build Your Plan

4.1 Add accounts.

4.2 Add planned transactions.

4.2.1 Income.

4.2.2 Bills.

4.2.3 Variable spending categories.

4.3 Generate projections to see expected balances and net.

## 5.0 Save A Budget From Projections

5.1 Generate projections.

5.2 Click Save as Budget.

5.3 Review the Budget grid.

5.3.1 Planned amounts can be edited.

5.3.2 Actual amounts can be recorded as the period progresses.

## 6.0 Project From Budget

6.1 After editing the Budget grid, click Project from Budget.

6.2 This regenerates projections using the budget occurrences as the source.

## 7.0 Track Actual Transactions

7.1 Use Actual Transactions to record what really happened.

7.2 Use this to compare planned vs actual and refine future budgets.

## 8.0 Troubleshooting

8.1 If budgets do not appear.

8.1.1 Confirm the selected workflow is Budget.

8.1.2 Generate projections before saving a budget.
