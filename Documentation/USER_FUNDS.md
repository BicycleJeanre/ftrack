# Funds Scenarios

## 1.0 Purpose

1.1 Explain how to use Funds scenarios to model a shared pool with ownership expressed as shares.

## 2.0 When To Use Funds

2.1 Use Funds when you want.

2.1.1 A pooled fund view with NAV.

2.1.2 Share-based ownership derived from contributions.

2.1.3 A scoped summary view across account types.

## 3.0 Create A Funds Scenario

3.1 Go to Forecast.

3.2 Create a scenario.

3.3 Set Type to Funds.

3.4 Set Period Type.

3.4.1 Month is typical.

## 4.0 Configure Accounts

4.1 Add the fund accounts.

4.1.1 Asset accounts that represent the pooled holdings.

4.1.2 Liability accounts when applicable.

4.1.3 Equity style accounts for investors when your model uses them.

4.2 Set starting balances to represent the initial state.

## 5.0 Model Contributions And Withdrawals

5.1 Use planned transactions to model contributions and withdrawals.

5.2 Use separate accounts to represent different investors when applicable.

5.3 Generate projections to see how NAV and ownership change.

## 6.0 Read The Funds Summary

6.1 Funds scenarios show a summary card section.

6.1.1 NAV equals Assets minus Liabilities.

6.1.2 Shares are derived from net contributions.

6.1.3 Ownership percentages are derived from shares.

6.2 Use the scope selector to view All, Asset, Liability, Equity, Income, or Expense.

## 7.0 Troubleshooting

7.1 If shares look unexpected.

7.1.1 Confirm transactions are categorized correctly.

7.1.2 Confirm investor contribution accounts are modeled consistently.
