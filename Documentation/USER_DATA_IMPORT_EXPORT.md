# Import And Export

## 1.0 Purpose

1.1 Explain how to export your FTrack data to a JSON file and import it again.

## 2.0 Where Your Data Lives

2.1 FTrack stores app data in browser storage.

2.2 Export is the primary way to create a portable backup.

## 3.0 Export Data

3.1 Use the navbar button Export Data.

3.2 Or use the shortcut.

3.2.1 Mac Cmd+E.

3.2.2 Windows Ctrl+E.

3.3 Your browser downloads a JSON file named like.

3.3.1 ftrack-backup-YYYY-MM-DD.json.

## 4.0 Import Data

4.1 Use the navbar button Import Data.

4.2 Or use the shortcut.

4.2.1 Mac Cmd+I.

4.2.2 Windows Ctrl+I.

4.3 Select a JSON export file.

4.4 Confirm the import.

4.4.1 Import replaces your current data.

4.4.2 The app reloads after a successful import.

## 5.0 Clear Data

5.1 Use the navbar button Clear Data.

5.2 Confirm the warning.

5.3 The app reloads after clearing.

5.4 Export first if you want a backup.

## 6.0 Troubleshooting

6.1 Import fails.

6.1.1 Confirm the file is valid JSON.

6.1.2 Confirm it was exported from FTrack.

6.2 You hit storage limits.

6.2.1 Export your data.

6.2.2 Clear old scenarios.
