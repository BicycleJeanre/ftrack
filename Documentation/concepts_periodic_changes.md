# Periodic Changes

## 1.0 Periodic Changes

## 1.1 Amount Adjustments Over Time

- **Flat increase**: Add a fixed amount each period such as 100 dollars annually.
- **Flat decrease**: Subtract a fixed amount each period.
- **Percentage increase**: Grow by a percentage such as 3 percent annually for salary raises.
- **Percentage decrease**: Shrink by a percentage.
- **Compound interest**: Apply exponential growth such as investment returns.

## 1.2 What Are Periodic Changes

Periodic changes apply adjustments to transaction amounts at regular intervals. Instead of manually changing amounts, you define a change rule that automatically applies each period.

## 1.3 Types Of Changes

### 1.3.1 Flat Increase

Add a fixed dollar amount each period. Example: Salary increases by 100 dollars every quarter.

- Useful for step raises or bonuses.
- Good when you know the exact dollar amount of the increase.

### 1.3.2 Flat Decrease

Subtract a fixed dollar amount each period. Example: Loan principal decreases by 500 dollars monthly.

- Useful for loan amortization models.
- Good for tracking declining expenses.

### 1.3.3 Percentage Increase

Apply a percentage increase each period. Example: Salary grows 3 percent annually.

- Useful for inflation adjustments.
- Good for modeling salary growth or investment returns.
- Works with any recurrence pattern.

### 1.3.4 Percentage Decrease

Apply a percentage decrease each period. Example: Expenses shrink 2 percent annually as you optimize.

- Useful for modeling cost reduction plans.
- Good for depreciation calculations.

### 1.3.5 Compound Interest

Apply exponential growth like investment returns or savings account interest. Example: Investment grows at 7 percent annually.

- Useful for investment and savings projections.
- Accounts for interest on interest effects.
- Best used with accounts rather than individual transactions.

## 1.4 Setting Up A Periodic Change

1. Create a transaction with recurrence.
2. Open the recurrence details.
3. Add a periodic change rule.
4. Select the change type, flat, percentage, compound.
5. Enter the amount or percentage.
6. Specify when changes apply such as annually or every quarter.
7. Save and generate projections.

## 1.5 Real World Examples

### 1.5.1 Salary With Annual Raises

- **Transaction**: Weekly paycheck 1,000 dollars
- **Recurrence**: Every Friday
- **Periodic Change**: 3 percent increase annually

### 1.5.2 Inflation Adjusted Expenses

- **Transaction**: Monthly groceries 400 dollars
- **Recurrence**: 1st of each month
- **Periodic Change**: 2.5 percent increase annually

### 1.5.3 Investment Growth

- **Transaction**: Monthly investment contribution 500 dollars
- **Recurrence**: 15th of each month
- **Periodic Change**: 7 percent compound growth annually
