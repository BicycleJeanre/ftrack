# Keyboard Shortcuts

## 1.0 Keyboard Shortcuts

## 1.1 Quick Actions

- **Save row**: Press Enter to save edits.
- **Delete row**: Press Delete to remove a row.
- **Add new row**: Cmd+Shift+A on Mac or Ctrl+Shift+A on Windows.
- **Open settings**: Cmd+, on Mac or Ctrl+, on Windows.

## 1.2 Grid Navigation

- Move to next cell: Mac Tab, Windows and Linux Tab
- Move to previous cell: Mac Shift+Tab, Windows and Linux Shift+Tab
- Move up: Mac Up Arrow, Windows and Linux Up Arrow
- Move down: Mac Down Arrow, Windows and Linux Down Arrow
- Edit cell: Mac Enter, Windows and Linux Enter
- Save cell: Mac Enter, Windows and Linux Enter

## 1.3 Row Management

- Add new row: Mac Cmd+Shift+A, Windows and Linux Ctrl+Shift+A
- Delete row: Mac Delete or Fn+Delete, Windows and Linux Delete
- Duplicate row: Mac Cmd+D, Windows and Linux Ctrl+D

## 1.4 Application

- Open settings: Mac Cmd+,, Windows and Linux Ctrl+,
- Export data: Mac Cmd+E, Windows and Linux Ctrl+E
- Import data: Mac Cmd+I, Windows and Linux Ctrl+I

## 1.5 Tips For Power Users

- Use Tab to move between cells quickly when entering data.
- Press Enter to save without navigating away.
- Use Cmd+Shift+A on Mac or Ctrl+Shift+A on Windows to add rows in bulk.
- Delete removes the entire row; be careful.
- Most shortcuts work in both grid and modal dialogs.
