Welcome to FTrack

A powerful financial forecasting tool for managing scenarios, accounts, and transactions.

Getting Started

  • Forecast: Create and manage financial scenarios with projections
  • Accounts: View and edit all accounts across scenarios
  • Scenarios: Compare different financial plans side-by-side
  • Transactions: Track planned and actual transactions
  • Projections: Generate automatic forecasts based on your data

Features

  • Modern Tabulator-based grids with inline editing
  • Scenario-based financial planning
  • Account management with periodic changes
  • Planned vs actual transaction tracking
  • Automated projection calculations
  • Multi-currency support
  • Dark theme interface
